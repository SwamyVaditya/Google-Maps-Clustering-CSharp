Example of adding Google Maps Clusteing to existing webforms project



- A new Webforms project has been created

- Some unused files from the new project has been deleted

- Custom web api controllers has been added

- The google maps clustering dll from MVC has been added to libraries folder

- The needed data has been copied to folders App_Data and App_Config

- The web.config has been setup with configSections

- The global.asax has been configured
